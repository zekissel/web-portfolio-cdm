Visual Argument


This is my Visual Argument, unofficially titled "Go Outside!". For this project, I utilized the software 'GIMP' to model a rhetoric that challenges our embrace of technology through images alone.

The project started as one hand-drawn .png image (the first frame), which carries the same message as the finished product: go outside. I developed the visual further by drawing more images and using GIMP to optimize the slideshow into a .gif. The final visual not only presents the original call to action, but also highlights the disconnect between humans and nature.
